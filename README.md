# 🏢 Mi-Admin

### Sistema Inteligente para la Administración de Alquileres, Propiedades y Edificios

**Administra propiedades de forma profesional, automatiza recibos, controla pagos y toma decisiones con analíticas en tiempo real.**

*Menos trabajo administrativo. Más control.*

</p>

---

## 📌 Descripción

**Mi-Admin** es un sistema de escritorio desarrollado para administradores de edificios, consorcios, complejos de departamentos y propietarios que necesitan gestionar alquileres de forma rápida, organizada y profesional.

El objetivo principal del proyecto es eliminar las tareas repetitivas mediante automatización inteligente, permitiendo administrar cientos de propiedades desde una única interfaz moderna.

Desde la generación automática de recibos hasta el análisis financiero mediante gráficos interactivos, Mi-Admin centraliza todo el proceso administrativo.

---

### 📥 Instalación

1. Haz clic en **"Descargar Mi-Admin"**.
2. Descarga el archivo desde Google Drive.
3. Descomprime el contenido si es necesario.
4. Ejecuta **Mi-Admin.exe**.
5. ¡Comienza a administrar tus propiedades!

> **No requiere conocimientos técnicos.** Solo descarga, ejecuta y empieza a trabajar.

## 📄 Generación Inteligente de Recibos

Mi-Admin puede generar automáticamente los recibos mensuales utilizando plantillas previamente configuradas.

El sistema combina:

* Conceptos generales del edificio
* Conceptos particulares del departamento
* Datos del inquilino
* Período correspondiente

Todo ello en un único proceso.

---

## ⚡ Automatización

Con un solo clic es posible:

* Crear todos los borradores del mes
* Editarlos antes de emitirlos
* Generar los recibos finales
* Exportarlos utilizando plantillas personalizadas de Excel o LibreOffice

---

## 📊 Dashboard Profesional

El Panel de Control incorpora gráficos de alto nivel para facilitar la toma de decisiones.

Incluye:

* Evolución mensual de ingresos
* Evolución de gastos
* Rentabilidad
* Cobrado vs Impago
* Indicadores financieros
* Estadísticas históricas

---

## 🔍 Auditoría Completa

Cada modificación importante queda registrada automáticamente.

Permite conocer:

* quién realizó una modificación
* cuándo ocurrió
* qué dato fue modificado
* historial completo de operaciones

Ideal para mantener trazabilidad y transparencia.

---

## 🗄 Gestión Global de Datos

Incluye un editor avanzado para realizar modificaciones masivas sobre la base de datos.

Funciones:

* edición directa
* filtros
* búsquedas
* guardar cambios
* revertir operaciones

---

## 🎨 Personalización

El sistema puede adaptarse completamente a cada administración.

Incluye:

* Logo personalizado
* Nombre comercial
* Tema Claro
* Tema Oscuro
* Plantillas propias de recibos
* Configuración de rutas
